# ESPN API Project

## Getting Started

1. **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

2. **Run the project:**
    ```bash
    python lineup.py
    ```

## Customizing for Your Team

- To use your own team, edit the `my_team` variable in `lineup.py - main()` and set it to your team's index or ID.
